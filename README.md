# README #

이 사이트는 파일을 웹에서 보거나 넘버링하는 하는 것을 제공합니다.

### What is this repository for? ###

* 파일 리더
* 만화 뷰
* 파일 넘버링
* Version: v1.0.0
* [Learn Markdown](https://bitbucket.org/tutorials/markdowndemo)

### How do I get set up? ###

#### File View

1. 폴더을 선택해줍니다.
2. Image, Text, PDF에 맞춰 보여줍니다.
3. 이전/다음 파일보기 또는 폴더보기가 가능합니다.
4. 텍스트일 경우 글씨크기가 조절 가능합니다.
   
#### Image View

1. 폴더를 선택해줍니다.
2. 폴더안의 이미지들을 보여줍니다.
3. 슬라이드, 만화형식(여백 유무) 선택하여 보기가 가능합니다.

#### Numbering

1. 폴더를 선택해줍니다.
2. 옵션에 따라서 설정을 하고 미리보기를 선택해줍니다.
3. 원하는 폴더에 결과를 저장해줍니다.
